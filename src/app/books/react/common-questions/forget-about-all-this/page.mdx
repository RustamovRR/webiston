## Bularni barchasini "Unuting"

**React Forget** — bu React ilovalarida memoizatsiyani avtomatlashtirishga qaratilgan yangi vositalar to'plami bo'lib, ehtimol, bundan keyin `useMemo` va `useCallback` kabi hook’larni ishlatish ortiqcha bo'lib qolar. Memoizatsiyani avtomatik ravishda boshqarish orqali _React Forget_ komponentlarning qayta render qilishini optimallashtiradi, foydalanuvchi bilan ishlash qulayligi(UX) va dasturchi bilan ishlash qulayligi(DX)ni yaxshilaydi. Ushbu avtomatlashtirish React’ning qayta render qilish xulq-atvorini obyekt identifikatori o'zgarishlaridan semantik qiymat o'zgarishlariga o'tkazadi, chuqur taqqoslashlarsiz, shunday qilib, ishlash samaradorligini oshiradi.

2021-yilda React Conf’da taqdim etilgan _React Forget_ hozirda "production"da umuman mavjud emas, lekin Meta’da Facebook, Instagram va boshqa platformalarda ishlab chiqarishda foydalanilmoqda va ichki ko'rsatkichlarga ko'ra "kutilgan natijalardan-da a'lo".
