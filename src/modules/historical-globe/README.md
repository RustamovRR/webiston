# Historical Globe - Interactive 3D Visualization

Tarixiy chegaralarni 3D globusda interaktiv ko'rsatish moduli.

## Xususiyatlar

- **3D Globe**: CesiumJS yordamida interaktiv 3D globus
- **Timeline**: BC 123,000 dan hozirgi kungacha tarixiy chegaralar
- **Animation**: Avtomatik animatsiya rejimi
- **Interactive**: Slider bilan istalgan yilga o'tish

## Texnologiyalar

- **CesiumJS**: 3D globe rendering
- **GeoJSON**: Tarixiy geodata
- **React**: UI components
- **Tailwind CSS**: Styling

## Ma'lumotlar manbai

[Historical Basemaps](https://github.com/aourednik/historical-basemaps) - tarixiy chegaralar geodata repositoriysi.

## Foydalanish

```bash
# Sahifaga kirish
http://localhost:3000/historical-globe
```
