# Muqaddima

Ushbu kitob JavaScript tilini va veb-brauzerlar hamda Node tomonidan amalga oshirilgan JavaScript API’larini qamrab oladi. Men uni dasturlash bo'yicha bir oz tajribaga ega bo'lgan va JavaScript’ni o'rganishni xohlayotgan o'quvchilar, shuningdek, JavaScript’dan foydalanayotgan, ammo bu tilni chuqurroq o'rganib, haqiqiy mutaxassis bo'lishni istayotgan dasturchilar uchun yozdim. Ushbu kitobdagi maqsadim JavaScript tilini har tomonlama va batafsil hujjatlashtirish, shuningdek, JavaScript dasturlarida mavjud bo'lgan muhim klient tomon (client-side) va server tomon (server-side) API’larga chuqur kirish taqdim etishdir. Natijada, bu kitob uzun va batafsilroq bo'lib chiqdi. Shunday bo'lsa-da, umid qilamanki, uni sinchiklab o'qish siz uchun foydali bo'ladi va bu kitobga sarflagan vaqtingiz dasturlash samaradorligining oshishi orqali o'zini oqlaydi.

Ushbu kitobning oldingi nashrlarida keng qamrovli ma'lumotnoma bo'limlari mavjud edi. Ammo hozirgi kunda bunday ma'lumotlarni bosma shaklda taqdim etish o'zini oqlamaydi, chunki dolzarb ma'lumotlarni onlayn tarzda tez va oson topish mumkin. Agar sizga JavaScript’ning asosiy yoki klient tomoniga oid ma'lumotlarni izlash kerak bo'lsa, [MDN veb-saytiga](https://developer.mozilla.org/) tashrif buyurishingizni tavsiya qilaman. Server tomonidagi Node API’lari uchun esa [Node.js’ning rasmiy hujjatlariga](https://nodejs.org/api/) murojaat qilishingizni maslahat beraman.

Mening muharririm Angela Rufinoga minnatdorchilik bildiraman: u meni reja bo'yicha to'g'ri borishimni ta'minlab turdi va kechikkan muddatlarimga sabr qildi. Shuningdek, texnik ko'rib chiquvchilarimga — Brayan Sletten, Elizabet Robson, Ethan Flanagan, Maksimiliano Firtman, Sara Vachs va Shalk Nitlingga rahmat. ## O'Reilly onlayn ta'lim

O'Reilly Media so'nggi 40 yildan ortiq vaqt davomida texnologiya va biznes bo'yicha treninglar, bilimlar hamda tahlillar orqali kompaniyalarning muvaffaqiyatga erishishiga yordam berib kelmoqda.

Bizning noyob ekspertlar va innovatorlar tarmog'imiz o'z bilim va tajribalarini kitoblar, maqolalar va onlayn o'quv platformamiz orqali baham ko'radi. O'Reilly’ning onlayn o'quv platformasi sizga talabga binoan jonli trening kurslari, chuqurlashtirilgan o'quv yo'nalishlari, interaktiv kodlash muhiti hamda O'Reilly va 200 dan ortiq boshqa nashriyotlarning keng matn va video kolleksiyasiga kirish imkonini beradi. Qo'shimcha ma'lumot uchun http://oreilly.com manziliga tashrif buyuring.
