## Umumiy keng tarqalgan savollar va ularga javoblar

Endi React nima qilishini va ichki ishlash mexanizmi qandayligini yaxshiroq tushunib olganimizdan so'ng, React ilovalarini yozishda uning amaliy qo'llanilishiga chuqurroq to'xtalib chiqmoqchimiz. Ushbu bobda React bo'yicha keng tarqalgan savollarga javoblarni o'rganib, memoizatsiya(memoization), kechiktirilgan yuklash(lazy loading) va ishlash samaradorligi(performance) atrofidagi tushunchalarni kengaytiramiz. Keling, avvalo memoizatsiya haqida gaplashamiz.
