# Funksiya ta'rifi ifodalari

*Funksiya ta'rifi ifodasi* JavaScript funksiyasini aniqlaydi, va bunday ifoda qiymati yangi belgilangan funksiya hisoblanadi. Qaysidir ma'noda, funksiya ta'rifi ifodasi "funksiyaning literali" bo'lib, obyekt boshlang'ich qiymati "obyekt literal" sifatida qaralganiga o'xshaydi.

Funksiya ta'rifi ifodasi odatda `function` kalit so'zi bilan boshlanadi, so'ngra qavslar ichida noldan boshlab bir nechta identifikatorlar (parametrlar nomi) vergul bilan ajratilgan holda keladi. Funksiya tanasi esa jingalak qavslar ({}) ichida yozilgan JavaScript kod blokidan iborat bo'ladi. Masalan:
``` js 
// Bu funksiya unga yuborilgan qiymatning kvadratini qaytaradi.
let square = function(x) { return x * x; };
```

Funksiya ta'rifi ifodasi funksiya uchun nomni ham o'z ichiga olishi mumkin. Shuningdek, funksiyalar funksiya ifodasi o'rniga funksiya operatori yordamida ham belgilanishi mumkin.

ES6 va undan keyingi versiyalarda funksiya ifodalari ixcham "strelka funksiyasi" (arrow function) sintaksisidan foydalanishi mumkin. Funksiyalarni aniqlashning to'liq tafsilotlari 8-bobda keltirilgan.
