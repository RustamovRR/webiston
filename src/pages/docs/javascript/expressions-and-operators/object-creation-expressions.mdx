# Obyekt yaratish ifodalari

*Obyekt yaratish ifodasi* yangi obyekt yaratadi va bu obyektning xossalarini ishga tushirish uchun funksiya (konstruktor deb ataladi) chaqiradi. Obyekt yaratish ifodalari chaqiruv ifodalariga o'xshaydi, faqat ular `new` kalit so'zi bilan boshlanadi:
``` js
new Object()
new Point(2,3)       
```

Agar obyekt yaratish ifodasida konstruktor funksiyaga hech qanday argument uzatilmasa, bo'sh qavslar juftligini tashlab ketish mumkin:
``` js
new Object
new Date     
```

Obyekt yaratish ifodasining qiymati yangi yaratilgan obyektdir. Konstruktorlar haqida batafsil ma'lumot 9-bobda batafsil yoritilgan.
