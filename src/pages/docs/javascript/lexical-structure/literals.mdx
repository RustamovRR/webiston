# Literallar

*Literal* - bu dasturda to'g'ridan-to'g'ri paydo bo'ladigan ma'lumot qiymati. Quyidagilar barchasi literallardir:
``` js 
12              // O'n ikki soni
1.2             // Bir butun o'ndan ikki soni
"hello world"   // Matnli satr
'Hi'            // Boshqa bir matnli satr
true            // Boolean qiymati
false           // Boshqa Boolean qiymati
null            // Obyektning yo'qligi
```

Son va starli literallari haqida to'liq tafsilotlar 3-bobda keltirilgan.
