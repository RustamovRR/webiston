# Izohlar

JavaScript izohlarning ikki turini qo'llab-quvvatlaydi. `//` va qator oxiri orasidagi har qanday matn izoh sifatida qaraladi va JavaScript tomonidan e'tiborsiz qoldiriladi. `/*` va `*/` belgilari orasidagi har qanday matn ham izoh sifatida qaraladi; bu izohlar bir necha qatorga cho'zilishi mumkin, lekin ichma-ich bo'lishi mumkin emas. Quyidagi kod qatorlari barchasi JavaScript’da to'g'ri izohlar hisoblanadi:

``` js
// Bu bir qatorli izoh.

/* Bu ham izoh */  // va bu yana bir izoh.

/*
 * Bu ko'p qatorli izoh. Har bir qator boshidagi qo'shimcha * belgilari
 * sintaksisning majburiy qismi emas; ular shunchaki chiroyli ko'rinadi!
 */
```
