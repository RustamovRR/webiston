# Identifikatorlar va Zaxiralangan so'zlar

*Identifikator* - bu oddiy nom. JavaScript’da identifikatorlar konstantalar, o'zgaruvchilar, xossalar, funksiyalar va klasslarni nomlash uchun hamda JavaScript kodidagi ma'lum sikllarga belgi berish uchun ishlatiladi. JavaScript identifikatori harf, pastki chiziq (`_`) yoki dollar belgisi (`$`) bilan boshlanishi kerak. Keyingi belgilar harf, raqam, pastki chiziq yoki dollar belgilari bo'lishi mumkin. (Raqamlar birinchi belgilar sifatida ishlatilmaydi, chunki bu JavaScript’ga identifikatorlar bilan sonlarni osonlik bilan ajratishga yordam beradi) Quyidagilar barchasi to'g'ri identifikatorlardir:
``` js
i
my_variable_name
v13
_dummy
$str
```

Har qanday til kabi, JavaScript ham ma'lum identifikatorlarni tilning o'zi tomonidan foydalanish uchun zaxiralab qo'yadi. Bu "zaxiralangan so'zlar" oddiy identifikatorlar sifatida ishlatilishi mumkin emas. Ular keyingi bo'limda keltirilgan.
