# Xulosa

Ushbu bobda JavaScript dasturlari qanday qilib eng quyi darajada yozilishi ko'rsatib o'tildi. Keyingi bob bizni bir pog'ona yuqoriga olib chiqadi va JavaScript dasturlari uchun hisoblashning asosiy birliklari sifatida xizmat qiluvchi primitiv turlar va qiymatlar (sonlar, satrlar va boshqalar) bilan tanishtiradi.
