# Unicode

JavaScript dasturlari Unicode belgilar to'plami yordamida yoziladi va siz har qanday Unicode belgilardan satrlar va izohlar ichida foydalanishingiz mumkin. Portativlik va tahrirlash qulayligi uchun odatda identifikatorlarda faqat ASCII harflari va raqamlaridan foydalaniladi. Lekin bu faqat dasturlash konvensiyasi bo'lib, til Unicode harflari, raqamlar va ideogrammalarni (lekin emojilarni emas) identifikatorlarda ishlatishga ruxsat beradi.  Bu dasturchilar matematik belgilar va ingliz tilidan boshqa tillardagi so'zlarni konstantalar va o'zgaruvchilar sifatida ishlatishi mumkinligini anglatadi:
``` js
const π = 3.14;
const sí = true;
```
