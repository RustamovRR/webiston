# JavaScript dasturining matni

JavaScript katta-kichik harflarni farqlaydigan(case-sensitive) tildir. Bu shuni anglatadiki, til kalit so'zlari, o'zgaruvchilar, funksiya nomlari va boshqa *identifikatorlar* har doim harflarning izchil katta-kichikligida yozilishi kerak. Masalan, `while` kalit so'zi "while" deb yozilishi kerak, "While" yoki "WHILE" emas. Xuddi shunday, `online`, `Online`, `OnLine` va `ONLINE` to'rtta turli o'zgaruvchi nomlardir.

JavaScript dasturlardagi tokenlar orasida paydo bo'ladigan bo'shliqlarni e'tiborsiz qoldiradi. Ko'p hollarda, JavaScript qator uzilishlarini ham e'tiborsiz qoldiradi (lekin istisno uchun §2.6 ga qarang). Dasturlaringizda bo'shliqlar va yangi qatorlardan erkin foydalana olganingiz uchun, siz dasturlaringizni kodni o'qish va tushunishni osonlashtiradigan tartibli va izchil tarzda formatlashingiz va cheklashingiz mumkin.

Oddiy bo'shliq belgisidan (`\u0020`) tashqari, JavaScript tab’lar, turli ASCII boshqaruv belgilari va turli Unicode bo'shliq belgilarini ham oq bo'shliq sifatida taniydi. JavaScript yangi qatorlar, karetka qaytishlari va karetka qaytishi/qator uzilishi ketma-ketligini qator tugatuvchilari sifatida taniydi.
