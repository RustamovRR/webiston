# Koʻp oʻlchovli massiv'lar

JavaScript haqiqiy koʻp oʻlchovli massiv'larni qoʻllab-quvvatlamaydi, lekin siz ularni massiv'lar ichidagi massiv'lar yordamida hosil qilishingiz mumkin. Massiv'lar massiv'idagi qiymatga murojaat qilish uchun shunchaki `[]` operatoridan ikki marta foydalaning. Masalan, `matrix` o‘zgaruvchisi sonlar massiv'laridan iborat massiv deb faraz qilaylik. `matrix[x]`'dagi har bir element — bu sonlar massiv'idir. Bu massiv ichidagi ma'lum bir songa murojaat qilish uchun siz `matrix[x][y]` deb yozasiz. Quyida ikki o‘lchovli massiv'dan ko‘paytirish jadvali sifatida foydalanilgan amaliy misol keltirilgan:
``` js
    // Koʻp oʻlchovli massiv yaratish
    let table = new Array(10);                  // Jadvalning 10 ta qatori
    for(let i = 0; i < table.length; i++) {
      table[i] = new Array(10);               // Har bir qatorda 10 ta ustun mavjud
    }

    // Massiv'ni qiymatlar bilan to'ldirish
    for(let row = 0; row < table.length; row++) {
      for(let col = 0; col < table[row].length; col++) {
        table[row][col] = row*col;
      }
    }

    // 5*7 ni hisoblash uchun koʻp oʻlchovli massiv'dan foydalanish
    table[5][7]  // => 35
```
