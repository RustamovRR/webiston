# Ko'p o'lchamli massivlar

JavaScript haqiqiy ko'p o'lchamli massivlarni qo'llab-quvvatlamaydi, lekin massivlar ichidagi massivlar yordamida ularni shunga o'xshatib yozish mumkin. Massivlar massivi ichidagi biror qiymatga murojaat qilish uchun shunchaki `[]` operatorini ikki marta ishlating.

Masalan, `matrix` o'zgaruvchisi sonlar massivlaridan iborat massiv deb faraz qilaylik. `matrix[x]`'dagi har bir element â€” bu sonlar massividir. Bu massiv ichidagi aniq bir songa murojaat qilish uchun siz `matrix[x][y]` deb yozasiz.

Quyida ikki o'lchamli massivdan ko'paytirish jadvali sifatida foydalanilgan aniq bir misol keltirilgan:

``` js
// Ko'p o'lchamli massiv yaratish
let table = new Array(10);                 // Jadvalning 10 ta qatori
for(let i = 0; i < table.length; i++) {
    table[i] = new Array(10);              // Har bir qatorda 10 ta ustun bor
}

// Massivni initsializatsiya qilish
for(let row = 0; row < table.length; row++) {
    for(let col = 0; col < table[row].length; col++) {
        table[row][col] = row*col;
    }
}

// 5*7 ni hisoblash uchun ko'p o'lchamli massivdan foydalanish
table[5][7] // => 35
```
