# Xulosa

Ushbu bob JavaScript massiv'larini, jumladan, siyrak massiv'lar va massiv'simon obyekt'lar haqidagi chuqur tafsilotlarni ham qamrab olgan holda, batafsil yoritib berdi. Bu bobdan olinadigan asosiy xulosalar quyidagilardir:

*   Massiv literallari to‘rtburchak qavslar ichida vergul bilan ajratilgan qiymatlar ro‘yxati sifatida yoziladi.
*   Alohida massiv elementlariga to‘rtburchak qavslar ichida kerakli massiv indeksini ko‘rsatish orqali murojaat qilinadi.
*   ES6'da kiritilgan `for/of` sikli va `...` yoyish operatori massiv'larni aylanib chiqishning ayniqsa foydali usullaridir.
*   `Array` sinfi massiv'lar bilan ishlash uchun boy metodlar to‘plamini ta’riflaydi va siz `Array` API bilan yaxshilab tanishib chiqishingiz lozim.
