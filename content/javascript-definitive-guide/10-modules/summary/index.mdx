# Xulosa

Modullikning maqsadi — dasturchilarga o‘z kodlarining implementatsiya tafsilotlarini yashirish imkonini berishdir, toki turli manbalardan olingan kod qismlari bir-birining funksiya yoki o‘zgaruvchilarini qayta yozib yuborishidan xavotirlanmasdan katta dasturlarga yig‘ila olsin. Ushbu bob uch xil JavaScript modul tizimini tushuntirib berdi:

*   JavaScript'ning ilk davrlarida modullikka faqat darhol chaqiriladigan funksiya ifodalaridan mohirona foydalanish orqali erishish mumkin edi.
*   Node JavaScript tilining ustiga o‘zining modul tizimini qo‘shdi. Node modullari `require()` yordamida import qilinadi va o‘z eksportlarini `exports` obyektining xususiyatlarini o‘rnatish yoki `module.exports` xususiyatini o‘rnatish orqali ta’riflaydi.
*   ES6'da JavaScript nihoyat `import` va `export` kalit so‘zlari bilan o‘zining modul tizimiga ega bo‘ldi va ES2020 `import()` yordamida dinamik importlarni qo‘llab-quvvatlashni qo‘shmoqda.
