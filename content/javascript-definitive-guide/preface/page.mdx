# So'zboshi

Ushbu kitob JavaScript tilini hamda veb-brauzerlar va Node tomonidan implementatsiya qilingan JavaScript API'larini qamrab oladi. Men uni ma'lum bir dasturlash tajribasiga ega bo'lib, JavaScript'ni o'rganishni istagan o'quvchilar uchun ham, JavaScript'dan allaqachon foydalanadigan, lekin o'z tushunchalarini yangi bosqichga olib chiqishni va tilni haqiqatan ham **mukammal o'zlashtirishni** xohlagan dasturchilar uchun ham yozdim.

Bu kitob bilan mening maqsadim — JavaScript tilini **har tomonlama va uzil-kesil** hujjatlashtirish hamda JavaScript dasturlari uchun mavjud bo'lgan eng muhim klient tomoni va server tomoni API'lariga chuqur kirish qismini taqdim etishdir. Natijada, bu uzun va batafsil kitobga aylandi. Umid qilamanki, bu kitobni sinchkovlik bilan o'qib chiqish uchun arziydi va siz uni o'qishga sarflagan vaqtingiz dasturlash unumdorligingizning oshirishiga o'z hissasini qo'shadi.

Ushbu kitobning oldingi nashrlarida keng qamrovli ma'lumotnoma bo'limlari kelardi. Endi men bu materialni bosma shaklda kiritishning ma'nosi bor deb o'ylamayman, chunki eng so'nggi ma'lumotnoma materiallarini onlaynda shunchalik tez va oson topish mumkin. Agar sizga JavaScript yadrosi yoki klient tomoniga oid biror narsani qidirish kerak bo'lsa, men sizga MDN veb-saytiga tashrif buyurishni tavsiya qilaman. Server tomonidagi Node API'lari uchun esa, men to'g'ridan-to'g'ri manbaning o'ziga murojaat qilishni va Node.js ma'lumotnoma hujjatlaridan foydalanishni maslahat beraman.
