# JavaScript dasturining matni

JavaScript katta-kichik harflarga sezgir (case-sensitive) tildir. Bu shuni anglatadiki, tilning kalit so'zlari, o'zgaruvchilari, funksiya nomlari va boshqa _identifikatorlari_ har doim izchil katta-kichik harflarda yozilishi kerak. Masalan, `while` kalit so'zi aynan "while" deb yozilishi kerak, "While" yoki "WHILE" deb emas. Xuddi shunday, `online`, `Online`, `OnLine` va `ONLINE` - bular to'rtta bir-biridan farq qiluvchi o'zgaruvchi nomlaridir.

JavaScript dasturlardagi tokenlar orasida keladigan bo'shliqlarni e'tiborsiz qoldiradi. Aksariyat hollarda, JavaScript qator uzilishlarini ham e'tiborsiz qoldiradi (lekin istisno uchun ยง2.6-bo'limga qarang). Dasturlaringizda bo'shliqlar va yangi qatorlardan erkin foydalana olganingiz uchun, siz dasturingizni kodni o'qish va tushunishni osonlashtiradigan tarzda, chiroyli va izchil formatlashingiz hamda xatboshilar qo'yishingiz (indent) mumkin.

Oddiy bo'shliq belgisidan (`\u0020`) tashqari, JavaScript tabulyatsiya, turli ASCII boshqaruv belgilari va bir nechta Unicode bo'shliq belgilarini ham bo'shliq belgisi (whitespace) sifatida tan oladi. JavaScript yangi qator, karetka qaytishi va karetka qaytishi/qator uzilishi ketma-ketligini qator yakunlovchilari sifatida qabul qiladi.
