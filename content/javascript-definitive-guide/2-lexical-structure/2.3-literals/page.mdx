# Literallar

_Literal_ - bu dastur kodida to'g'ridan-to'g'ri yozilgan ma'lumot qiymatidir. Quyidagilar barchasi literallardir:

```js
12              // O'n ikki soni
1.2             // Bir butun o'ndan ikki soni
'hello world'   // Matnli satr
'Hi'            // Yana bir satr satr
true            // Mantiqiy (boolean) qiymat
false           // Ikkinchi mantiqiy qiymat
null            // Obyektning mavjud emasligi
```

Sonli va satrli literallar haqidagi to'liq tafsilotlar 3-bobda keltirilgan.
