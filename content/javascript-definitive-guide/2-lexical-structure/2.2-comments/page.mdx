# Izohlar

JavaScript ikki xil uslubdagi izohlarni qo‘llab-quvvatlaydi. `//` belgisidan boshlab qator oxirigacha bo'lgan har qanday matn izoh sifatida qaraladi va JavaScript tomonidan e'tiborsiz qoldiriladi. `/*` va `*/` belgilari orasidagi har qanday matn ham izoh hisoblanadi; bu turdagi izohlar bir necha qatorga cho'zilishi mumkin, lekin ichma-ich joylashtirilishi mumkin emas. Quyidagi kod qatorlari JavaScript’dagi to'g'ri izohlarga misoldir:

```js
// Bu bir qatorli izoh.

/* Bu ham izohdir. */ // va mana bu ham boshqa bir izoh.

/*
 * Bu ko'p qatorli izoh. Har bir qator boshidagi qo'shimcha * belgilari
 * sintaksisning majburiy qismi emas; ular shunchaki chiroyli ko'rinish beradi!
 */
```
